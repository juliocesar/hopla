Hopla

Hopla is not a utility belt. It's forged with the blood of a thousand enemies, and delivers deadly efficiency in trained hands.

Hopla runs as a Rake task. You can customise it, like the warrior-programmer you are. There are no hidden layers or anything of the sort. Read the source, and you'll understand what it does.

To get Hopla, run: 

$ wget https://raw.github.com/juliocesar/hopla.io/master/Rakefile

Then run the following command and follow the instructions: 

$ rake hopla

Hopla will output what it compiles in assets/ into the public/ directory. It features a built-in web server for use in development, but deploy as you see fit for real combat.

Hopla was forged with the fury of a thousand suns by Julio, and lives in GitHub.
